常识

  1. anaconda And conda 常见误区(英文) 中文翻译
  • anaconda: python的一个发行版本（即各种软件包的集合）
  • conda : 包管理软件和环境管理器 conda 的详细介绍

  1. conda and pip > pip可以允许你在任何环境中安装python包，而conda允许你在conda环境中安装任何语言包（包括c语言或者python）

  2. jupyter notebook 中添加conda新环境 >python -m ipykernel install --user --name your_env_name --display-name your_env_name jupyter notebook 添加 conda 环境
